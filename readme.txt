调用：python .\风险归因方案.py --w 156 --end-date --output-excel --output-chart --market-file --duration-file --product-file
环境依赖见requirement.txt

--w 数据长度（周），必输。用户在6.1.2 风险因子归因模型 输入
--end-date 无需输入，模型默认当日
--output-excel 需要在风控系统前端展示的返回信息。不指定位置时默认不导出，但命令行也会print。
--output-chart 需要在风控系统前端展示的返回信息。不指定位置时默认不画图。
--market-file  输入数据。数据格式见excel。风控系统每日终提供，模型计算四因子收益率序列及因子相关性矩阵并返回展示
--duration-file 输入数据。数据格式见excel。风控系统每日终提供，模型计算四因子收益率序列及因子相关性矩阵并返回展示
--product-file 输入数据。用户在5.9.1 总体风险归因分析（风险因子）下拉选择投组/产品，风控系统把该投组/产品的每日估值序列实时发给模型，模型进行回归-方差分解-风险归因并返回展示

返回信息的展示位置：
四因子收益率序列、因子相关性矩阵：6.1.2风险因子归因模型
回归系数表：5.9.1 总体风险归因分析（风险因子）-模型结果
因子方差贡献分析表、风险归因结果.png：5.9.1 总体风险归因分析（风险因子）-风险归因结果
回归统计表、方差分析表、回归系数表、协方差矩阵、方差分解表：5.9.1 总体风险归因分析（风险因子）-模型详情
